appkg

An appkg (a'package) app is just a Web App embedded in a SQLite database with optional OpenPGP encryption. The appkg host is an open source standalone app (that you can host yourself) and offers a barely functional built-in code editor!

appkg uses OpenPGP.js and sql.js.

Create a new app or try one of these examples:

  • todo.appkg - Simple ToDo list
  • secrets.appkg - Encrypted ToDo list (password: password)
  • passwords.appkg - Basic password manager. Remember to encrypt it!
